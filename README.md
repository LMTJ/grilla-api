grilla-api
======
REST API for La Grilla

## Getting Started
1. Checkout the repo

2. Install dependencies with `npm install`

3. Start application with `npm start`

## Tools

#### GUI
```bash
npm run gui
```

#### CLI
```bash
slc
```

## Dependencies
- [NodeJS](http://nodejs.org/)
- [Strongloop](https://strongloop.com/)
- [MariaDB](https://mariadb.org/)
